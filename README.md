# Story Teller - Web Card Game
### [EN] 
Strategic card game developed as a graduation project at KIC.
- **Tech:** PHP, JavaScript, CSS, HTML.
- **Key Feature:** Complex game logic, turn-based system, and server-side validation.

専門学校（KIC）の卒業制作として開発した戦略的カードゲームです。
- **技術スタック:** PHP, JavaScript, CSS, HTML.
- **特徴:** 複雑なゲームロジック、ターン制バトルシステム、サーバー側での勝敗判定の実装。
